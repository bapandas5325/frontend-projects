# 🧠 Real-Time RSS News Monitoring System

A lightweight and scalable **RSS-based news monitoring system** designed for real-time tracking, filtering, and classification of news data for **cyber cells, incident monitoring teams, research, and analysis units**.

This system focuses on **fast RSS ingestion, keyword-based matching, and structured news categorization with real-time dashboard visibility**.

---

## 🚀 Key Features

### 📡 RSS Feed Ingestion System
- Supports multiple RSS feeds (500+ scalable structure)
- Continuous polling for near real-time updates
- XML parsing and normalization of news content
- Structured extraction of title, description, source, timestamp, and news type

---

### 🏷️ Rule-Based News Classification
News is categorized using deterministic rules:

- Crime
- Political
- Job / Recruitment
- Entertainment
- Cyber / General incidents

**Classification is based on:**
- Keyword presence
- Pattern matching
- Context-based rules (not AI-based)

---

### 🗺️ Location Tagging System
- State-level tagging (e.g., West Bengal, Maharashtra)
- District-level detection (when available in RSS text)
- Rule-based keyword mapping system
- Enables region-specific monitoring and filtering

---

### ⏱️ Time-Based Filtering System
Users can filter news by time range:

- 1 Hour (live monitoring)
- 6 Hours
- 1 Day
- 7 Days
- 30 Days

Supports fast retrieval of recent and historical data.

---

### 📊 Dashboard Features
- 📡 Live news feed updates  
- 🏷️ Category filtering (Crime / Political / Jobs / Entertainment)  
- 🗺️ State-wise filtering  
- 📍 District-level filtering (where available)  
- ⏱️ Time-range filtering  
- 🔄 Manual refresh button  
- 📌 Highlighted important incidents
